# 🚀 CreatorHub

A simple productivity web app for content creators to manage ideas, schedule content, and track performance.

---

## 🌐 Live Website
👉 https://bluefd.github.io/CreatorHub/


---

## 🎯 Purpose
CreatorHub is built to help content creators:
- Organize content ideas
- Plan posting schedules
- Track content performance in a simple way

The goal is to improve consistency and productivity.

---

## ✨ Features

### 💡 Idea Manager
- Add new content ideas
- Delete ideas
- Simple idea tracking system

### 📅 Scheduler
- Plan content posting schedule
- Organize content timeline

### 📊 Basic Analytics
- Track total ideas
- Simple performance overview

### 💾 Data Storage
- Uses LocalStorage
- Data saved in browser automatically

---

## 🎨 UI Design
- iOS-inspired interface
- Glassmorphism style cards
- Smooth hover animations
- Clean and minimal layout
- Responsive design (mobile friendly)
